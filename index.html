Only Human

Only Human is rather clearly an odd one out in this list (being not software). It is a simple tabletop RPG ruleset, for when you don't want your players' characters to have magic powers. It is very heavily inspired by the Dead Simple RPG games (particularly blaster), but presentationally and mechanically changed slightly to a system / format that I prefer.

The current rules document includes the rather bare bones game rules (character creation, tests / skill checks, actions, combat), an example of how that could be implemented (the equipment and player roles that flesh out a game in a sci-fi setting), and some tips on how to run a game with core rules that fit on a single page.

Sweetrolls

Sweetrolls is a personal project that I have taken on, that aims to be "a web app for rolling dice with friends", that is it aims to facilitate verfied, shareable results to reasonably arbitrary dice rolls. It currently has two versions, a quickly built MVP version, and a slowly growing react version .

The former has limited dice rolling functionality, and no actual user validation, but it does allow "rooms" which multiple people can view at the same time. The latter has an entire (as of yet undocumented) dice rolling / math language defined for it, user authentication, and editable rooms and buttons, but it does not yet have the ability for multiple users to view the same room.
